¡Bienvenido al sistema de guardabosques!

Este es el sistema para administración de Guardabosques.

En el menú izquierdo posee las siguientes funciones:

Inicio: permite volver a esta página.

Gestionar usuarios: para agregar nuevos usuarios Guardabosques.

Jornadas: permite aprobar o rechazar jornadas de usuarios Guardabosques.

Búsqueda: para buscar usuarios Guardabosques de acuerdo a ciertos criterios.

Actividades: permite administrar las actividades que pueden realizar los usuarios Guardabosques.

Asistencias: permite aprobar o rechazar las asistencias de los usuarios Guardabosques.

Notificaciones: para enviar notificaciones a los usuarios Guardabosques.

Cerrar sesión: para finalizar la sesión en el sistema.
